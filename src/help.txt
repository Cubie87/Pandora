`help`: shows this command
`ping`: ping the bot
`roll` (`r`): roll some dice
`ctftime` (or `ctf`): grab some information about a CTF from ctftime
`join`: join the voice channel you're currently in
`leave`: leave the connected voice channel in the server
`play` (`p`): play some music (control with `pause`/`resume`/`stop`)
`grab`: reply with the audio file attached as the message
`chat`: chat with the bot
